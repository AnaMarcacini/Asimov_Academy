# Game Sales Dash

Dashboard para análise de vendas de video games (1980-2017) feito com Python.

## 🔧 Funções

- Ler e dispor dados de forma visual a partir do dataset
- Filtrar por ano os dados para uma análise mais minuciosa
- RangeSlider para um filtro de data específico
- Possibilidade de alternar entre o tema claro e escuro do layout
